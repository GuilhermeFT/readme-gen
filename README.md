To clone and run the project locally, follow the instructions below:

1. Clone the repository:
   ```
   git clone https://github.com/GuilhermeFT/readme-gen.git
   ```

2. Navigate to the project directory:
   ```
   cd readme-gen

4. Start the project:
   ```
   npm start
